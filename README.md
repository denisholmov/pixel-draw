# Pixel Draw

Рисование на экране жестами руки через веб-камеру.

## Запуск

```bash
git clone https://github.com/denisholmov/pixel-draw.git
cd pixel-draw
chmod +x setup.sh && ./setup.sh
```

Windows: `setup.bat`

Установка с нуля (Python, git, всё нужное) через Cursor: [INSTALL_PROMPT.md](INSTALL_PROMPT.md)

## Жесты

| Жест | Действие |
|------|----------|
| ☝️ Указательный вверх, остальные сжаты | Рисование |
| ✊ Кулак | Стирает |
| 🖐 Ладонь раскрыта | Пауза |
| **Q** / **C** | Выход / очистить |
